# 環境構築

## nvmのインストール
```terminal
git clone git://github.com/creationix/nvm.git ~/.nvm
```

## JShintのインストール
```terminal
npm install -g jshint
```

## NeoBundleのインストール
```terminal
mkdir -p ~.vim/bundle
git clone https://github.com/Shougo/neobundle.vim ~/.vim/bundle/neobundle.vim
```
vimの画面でNeoBundleのプラグインをインストール
```vim
:NeoBundleInstall
```
